Use Case Analysis

Path 1

  1. User Opens Facebook Messenger App
  2. User Auto Logs in as they are already logged into the Facebook App
  3. User Searches for friend “Joey” to send a message to
  4. User taps on the appropriate search result
  5. User Types text message, which includes a plethora of emojis and sends it
  6. User follows up text message with a GIF of Donald Trump
  7. The user switches out of the app

 

Path 2

  1. User Opens Facebook Messenger App
  2. User manually logs in in as they are not already logged into the Facebook App
  3. User navigates to the call history page
  4. User sees that they most recently called their mother
  5. User calls their mother again
  6. After a 33 minute phone conversation with their mother they end the conversation
  7. During the conversation the user receives multiple messages and a call from a friend which they select to press ignore on
  8. The user navigates back to the messages page to reply to the messages they received
  9. The user logs out

 

Path 3

  1. User Opens Facebook Messenger App
  2. User does not need to log in as they are already logged in
  3. User navigates to the payments page
  4. User selects to send $30
  5. User selects “Joey” as the recipient of the $30
  6. User Confirms the transaction
  7. The transaction is completed
  8. User switches out of the messenger app
