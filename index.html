An Online Portfolio

This site showcases the design carried out in SYDE 542 using the Massive Template.

Portfolio Goals

The goal of this portfolio is to showcase methodologies learned in SYDE 542: User Intercace Design. There are two primary projects which will be developed using these techniques. The first of which is an Enterprise Dashboard created for Users of our fourth year design project. This dashboard will be used by ad managers in order to view the performance of their ads and allow for ad delivery management. The second project is a redesign of the mobile Facebook Messenger application using approaches discussed in SYDE 542.
